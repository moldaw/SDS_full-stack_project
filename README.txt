This is a simple web app for creating and maintaining a movie watchlist, created by using a MERN stack for the SDS full-stack course.

The app uses MERN to maintain a MongoDB database of added movies, React for frontend UI elements, Node for the backend logic and Express for the middleware communication. In the app the user can add a movie they want to watch, check it as watched or uncheck it, or delete the movie from the register.

To run the project, make sure MongoDB, Node.js and npm are installed. Then clone this repository, run "npm install" on both backend and frontend folders, and finally run "node server.js" on the backend folder and "npm start" on the frontend folder. This should automatically open the app for you, but if it doesn't it can be found in your browser at http://localhost:3000 and the DB API at port http://localhost:5000/api/movies

This repository contains the project itself, a learning diary documenting my thoughts on this course, and a video showcasing the app on my browser.
